# 🚗 SafeCar - Ανάλυση, Σχεδίαση & Υλοποίηση Πληροφοριακού Συστήματος

Το project αυτό αποτελεί την ολοκληρωμένη εργασία της **Ομάδας 9** στο μάθημα **Ανάλυση και Σχεδίαση Συστημάτων**. Αντικείμενο του έργου είναι η ανάλυση, ο σχεδιασμός και η ανάπτυξη ενός νέου Πληροφοριακού Συστήματος για το πρακτορείο ασφαλειών οχημάτων **"SafeCar"**, με σκοπό την ψηφιοποίηση και αυτοματοποίηση των επιχειρηματικών του διεργασιών (BPA).


## 📌 Περιγραφή Έργου & Λειτουργικότητα
Το σύστημα αντικαθιστά τις παραδοσιακές χειρόγραφες διαδικασίες του πρακτορείου με ένα σύγχρονο ηλεκτρονικό περιβάλλον, επιλύοντας προβλήματα καθυστερήσεων και έλλειψης οργάνωσης.

### Κύριες Επιχειρηματικές Διαδικασίες (Features):
1. **Διαχείριση πακέτων ασφάλισης:** Διαπραγμάτευση και καταγραφή πακέτων από τον διευθυντή.
2. **Νέα Ασφάλιση οχήματος πελάτη:** Ηλεκτρονική καταγραφή στοιχείων, επιλογή πακέτου και έκδοση συμβολαίου σε συνεργασία με την ασφαλιστική εταιρεία.
3. **Ανανέωση ασφάλειας οχήματος:** Ταχεία διαδικασία επικοινωνίας και ανανέωσης συμβολαίων με τους πελάτες.
4. **Διαχείριση ζημιών και αποζημιώσεων:** Ηλεκτρονική διασύνδεση με πραγματογνώμονες και τροχαία για την έκδοση πορισμάτων και την απόδοση αποζημιώσεων.

## 🛠️ Τεχνολογίες & Εργαλεία (Tech Stack)
* **Γλώσσα Προγραμματισμού:** Java (Υλοποίηση του τελικού διαγράμματος κλάσεων και του επιχειρησιακού σεναρίου).
* **Περιβάλλον Ανάπτυξης (IDE):** Eclipse 2023-09 (4.29.0).
* **Εργαλεία Μοντελοποίησης:** Visual Paradigm (Σχεδίαση UML διαγραμμάτων AS-IS, TO-BE, Περιπτώσεων Χρήσης, Κλάσεων, Αντικειμένων και Ακολουθίας).
* **Λειτουργικό Σύστημα Ανάπτυξης:** Windows 11.
